# 🪔 Diwali Sales Analysis

## 📌 Project Overview
This project analyzes **Diwali sales data** to uncover customer purchasing behavior, product performance, and demographic insights. The goal is to help businesses understand festive season trends and optimize marketing strategies for future campaigns.

## 🎯 Objectives
- Explore sales patterns during the Diwali festival.
- Identify top-selling product categories and regions.
- Understand customer demographics (age, gender, occupation, etc.).
- Provide actionable insights for business growth.

## 📂 Dataset
- **Source:** Diwali sales dataset (CSV format).
- **Features include:**
  - Customer demographics (Age, Gender, Occupation, State)
  - Product details (Category, Item Purchased)
  - Purchase amount

## 🛠️ Tools & Technologies
- **Python** 🐍
- **Libraries:**
  - `pandas` – data cleaning & manipulation
  - `numpy` – numerical analysis
  - `matplotlib` & `seaborn` – data visualization
  - `scikit-learn` – optional ML insights

## 📊 Key Insights
- Which product categories perform best during Diwali.
- Spending behavior across different age groups and occupations.
- Regional differences in purchase amounts.
- Gender-based purchasing trends.
